Gyazobank
=========

[![Deploy](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy)

Private Gyazo Server


Requirements
------------

* coffee-script
* Amazon S3 (or else)


Configurations
--------------

Set environment variables:

* `AWS_S3_ENDPOINT` (Example: `s3.amazonaws.com`)
* `AWS_S3_ACCESS_KEY`
* `AWS_S3_SECRET_KEY`
